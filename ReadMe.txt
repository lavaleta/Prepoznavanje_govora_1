The code is Python 3.7 and Python 3.8 compatible.
NOTE: In Python 3.8 a warning appears about plot, yet the program will compile as intended.

The program supports multiple channels.
If there is more than one channel, it takes only the first one in consideration.

To show Histogram and Scectrogram data has to be entered first (Enter data).

To use some .wav files that are not included, place them in /data folder.
